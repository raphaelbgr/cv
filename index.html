How to Use

The HTML versions render as a designed two-page A4 layout — open in Chrome, then Ctrl+P → Save as PDF, with margins set to None (the page has its own A4 margins built in). Send the resulting PDF to recruiters and hiring managers.

The Markdown versions are plain text optimised for ATS parsers — paste into Workday, Greenhouse, LinkedIn Easy Apply, and similar online application forms where free-form text upload is required. Identical content, no styling that confuses embedding-based ranking.

The keyword surface across all four variants reflects a 2026 JD-coverage audit against live senior streaming/CTV postings — see jd-coverage-audit.md for the full analysis.
